# Genre Prompt Library

Use this file when the user wants a specific music style or asks for richer Suno wording. These entries are curated from the user's Downloads materials, especially `suno曲风提示词.docx`, then tightened for practical Suno use.

## Prompt Formula

Use 4-7 descriptors:

`genre/subgenre, tempo or groove, key instruments/sound design, vocal identity, production texture, energy arc`

Add BPM only when it helps the genre. Do not stack many unrelated subgenres.
